# End-to-End Data Engineering Project using AWS + Apache Iceberg + Snowflake

This project implements an ETL pipeline for data extraction, transformation, and loading using AWS, Apache Iceberg, and Snowflake.

## Folder Structure

- `data/`: Stores raw, processed, and log files.
- `scripts/`: Contains ETL pipeline scripts.
- `config/`: Configuration files for the pipeline.
- `tests/`: Unit and integration tests.
- `notebooks/`: Jupyter notebooks for exploration and prototyping.

## Setup

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
